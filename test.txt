testing git!

instruction:
1) Проинициализировать git локально (git init)
2) добавить файлы (git add .)
3) Закоммитить файлы (git commit -m "comment")
4) Создать удаленный репозиторий
5) Подключить локальный репозиторий к удаленному 
   (git remote add origin https://github.com/Ishaak-web/<навание репы в гитхабе>з.git)
6) для первого коммита на ветке (git push -u origin <название ветки>),
   а для полсдеующих на ту же ветку просто git push
